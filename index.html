Laravel ile Alıntılar Projesi.

Projenin oluşturulma amacı laravel framework üzerine kendimizi geliştirmek ve neler yapılabildiğini görmek, son hali düzenlenmiş halidir. Proje geliştirilirken bir çok özellik kullanılıp daha sonra projeyle alakalı olmadığı için kaldırılmıştır. Projeyi git üzerinden çekebilmek için ( Git kurulu olduğunu varsayıyoruz );

$ git clone https://github.com/kemalaydin/Alintilar.git

Projeyi çalıştırabilmek için ;

$ cd /dosya_yolu
$ cd composer update

veritabanını artisan migration komutuyla oluşturabilirsiniz.

App/Config dosyasında ki database.php dosyasını kendinize göre düzenleyin
